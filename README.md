## Inclusion policy

An item is retained only when it strengthens technical capability, architectural judgment, research awareness, enterprise applicability, career leverage, or risk awareness. Funding-only stories, unsupported claims, generic commentary, duplicate reports, and temporary hype are excluded.

## Evidence policy

| Grade | Standard |
|---|---|
| A | Primary source with reproducible evidence, code, data, or specification |
| B | Primary source with a clear technical methodology |
| C | Reputable secondary analysis grounded in traceable evidence |
| D | Community observation; watchlist only |
| E | Unverified; excluded |

Each factual table row must provide a primary citation wherever one exists. Event time, publication time, and collection time are recorded separately.
